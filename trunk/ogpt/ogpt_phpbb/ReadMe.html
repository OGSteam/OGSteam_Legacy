Instructions d'installation d'OGame Portail Team !

Instructions pour l'installation de OGPT beta 1.0.

Pré-requis

  • un serveur web (de préférence Apache).
  • PHP 4.1.0 ou supérieur (PHP 5 inclus).
  • MySQL 3.23.17 ou supérieur.

MySQL : OGPT ne fonctionnera pas correctement avec une version antérieure à la 3.23.17.

Installation

  1. Copier/envoyer sur votre serveur tous le contenu du répertoire upload/ dans le répertoire où vous souhaitez mettre vos forums.
  2. Lancez l'instalation du forum dans le dossié uploadé. Suivez les instructions (attention à bien lire ce qu'il y a de marqué).

Merci de signaler tous problème que vous puissiez rencontrer pendant l'installation sur les forums de OGPT.
